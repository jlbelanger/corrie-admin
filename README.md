# Corrieography Admin

## Development

### Setup

``` bash
git clone https://github.com/jlbelanger/corrie-admin.git
cd corrie-admin
./setup.sh
yarn start
```

View the site at http://localhost:3000/admin

### Lint

``` bash
yarn lint
```

### Test

``` bash
yarn test:cypress
```

## Deployment

``` bash
./deploy.sh
```
